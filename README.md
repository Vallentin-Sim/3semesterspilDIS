Dette projekt er baseret på en given opgave. 

Denne opgave handler om at lære, hvordan man i et distribueret system kan anvende synkronisering i form af et spil med flere spillere over det samme netværk.

Regler: +1 point for hver gang man bevæger sig. -1 Hver gang man går ind mod en væg. +10 hvis man går ind i en spiller, hvor modstanderen mister -10 point hver gang man går ind i dem. 

Server klassen håndtere trafikken og sikre logikken og synkroniseringen blandt brugerne. 

Klienten køres fra App'en. 

Mulige tilføjelser man kunne lave: 
  1. Skyde-system? (Shooter)
  2. Tilfældig allokeret spawns.
  3. Mulig sprite fix der kan fremkomme.
  4. Andet.
